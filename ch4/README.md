# Basic CRUD Application (Node.js + MySQL)

This is a simple full-stack CRUD project built using Node.js, Express, MySQL, and basic frontend (HTML, CSS, JavaScript).

The main goal of this project was to understand how frontend, backend, and database work together.

---

## Features

- User Registration
- View User Data
- Update Password
- Delete User
- Password hashing using bcrypt
- Basic form validation on frontend

---

## Technologies Used

- HTML
- CSS
- JavaScript
- Node.js
- Express.js
- MySQL
- bcryptjs
- fetch API

---

## How it works

- Frontend collects user input
- Data is sent to backend using fetch API
- Backend processes the request using Express
- Data is stored and managed in MySQL database
- Passwords are stored in hashed form using bcrypt

---

## API Endpoints

- POST /postdata → Register user  
- GET /userdata → Get all users  
- PUT /updatedata → Update password  
- DELETE /deletedata → Delete user  

---

## What I learned

- How to connect frontend and backend
- How REST APIs work
- How CRUD operations work with database
- Basic password security using bcrypt
- Handling async operations in JavaScript
